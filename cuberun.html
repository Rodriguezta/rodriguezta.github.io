Cube Run

Cube Run is an arcade runner game packed with action where players have to develop their skills as they progress.

The game was developed in Unity3D and published on the App Store as part of a Publishing task for my HND.

For this project we had to make a publishable game from scratch, adding monetization and making it a minimum viable product. For that we assembled a team of three people

My main tasks where:

- UI design & implementation.

- Menu navigation implementation.

- Audio design & implementation.

As already said, the game was published in the Google Play Store but unfortunately it can no longer be downloaded.
