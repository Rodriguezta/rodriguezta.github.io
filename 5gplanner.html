5G Private Network Planner

The 5G Planner was an application built in the Unity Engine, designed to build environments and calculate 5G coverage on private and public networks. The application contains multiple industry focused features like customizable 5G transmitters and network receptors and even a "Sims-like" digital twin, built from the ground up to facilitate environment digitalization.

My main tasks:

- Client-server arquitecture using TCP, allowing data calculation on Fivecomm's servers.

- Complete design and implementation of UI elements and UI structure using the ImGui library.

- Python script integration for image and document generation.

- 3D asset integration and optimization within the engine.

- Tools development.

- Debugging core features.

For more information: 5G Planner's website
