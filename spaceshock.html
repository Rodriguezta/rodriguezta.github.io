SpaceShock

SpaceShock is a fast paced infinite shoot-em up with randomly generated waves.

The game was developed in Unity3D as part of a Games Design task for my HND.

For this project we had to take a really early and un-funny prototype for the game and make it as fun as possible in a week's time. For that purpose we could do anything with the game itself, including changing all the mechanics, adding sound and visual effects and so on.

In this case I did the following:

- Added sound effects and vfx to almost every action for the player.

- Added a score system and visual/sound feedback for it.

- Added a combo system to go with the score system.

- Added a health system and visual/sound feedback for it.

- Created 6 new enemy types and their attack patterns.

- Modified the player movement and shooting mechanics.
